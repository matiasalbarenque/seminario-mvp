# UADE | TPO - Trabajo Final / Seminario - MVP
### Tecnologías: React, Typescript, Vite, Tailwind CSS, Shadcn

---

## Instalación de dependencias:

```bash
npm install

```

## Iniciar server de desarrollo

```bash
npm run dev
```

## Compilación

```bash
npm run build
```
